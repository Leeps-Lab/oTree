# LEEPS Lab oTree Experiments

## Quick start

```
> git clone https://github.com/Leeps-Lab/oTree.git
> cd oTree
> virtualenv -p `which python3` venv
> source venv/bin/activate
> pip install -r requirements.txt
> otree resetdb
> otree runserver
```
